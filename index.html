First-Time Setup — Firebase Config
Monologue uses a free Google Firebase backend for real-time messaging and image storage. This is a one-time step — once saved, you won't need to do this again on this browser.
  1. Go to console.firebase.google.com → Create a project (free)
  2. In the left menu: Build → Realtime Database → Create database → Start in test mode
  3. Project Settings (gear icon) → Web app (</>) → Register app → copy the config object
  4. Paste the config below and click Save
